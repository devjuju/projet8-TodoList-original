# ToDo & Co - original

[![Codacy Badge]](https://app.codacy.com/gh/devjuju/projet8-TodoList-original/dashboard)

## 📌 Contexte

Ce dépôt représente l'état initial de l'application ToDo & Co.

Les travaux réalisés sur ce dépôt avaient pour objectif de :

mettre en place un environnement de développement reproductible avec Docker ;
vérifier le bon fonctionnement de l'application d'origine ;
réaliser un audit de qualité du code ;
réaliser un audit de performance afin d'identifier les principaux axes d'amélioration.

Les analyses effectuées ont permis d'établir un état des lieux de l'application avant toute évolution. Elles servent de référence pour mesurer l'impact des améliorations apportées par la suite.

La migration vers Symfony 3.4 LTS, les corrections fonctionnelles, l'amélioration de la sécurité, l'ajout des tests automatisés, la documentation technique ainsi que les audits finaux ont été réalisés dans un second dépôt consacré à la modernisation de l'application.

Ce dépôt constitue donc la base de comparaison (« avant ») utilisée pour évaluer les bénéfices des évolutions apportées au projet.
