# Docker - Projet original

## Objectif

Permettre l'exécution du projet Symfony 3.1 dans un environnement isolé.

## Configuration

- PHP 7.4 Apache
- MySQL 5.7
- phpMyAdmin

## Services Docker

| Service    | Port |
| ---------- | ---- |
| PHP Apache | 8067 |
| MySQL      | 3307 |
| phpMyAdmin | 8068 |
